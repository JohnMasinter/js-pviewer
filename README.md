# pviewer
This is a simple self-contained JavaScript picture viewer.  
Includes scrolling, zoom in/out, zoom-to width/height, image dimensions.  

## Files
v0 - simplest first rev of pviewer.  
- index.html   - tiny test html as example to call pviewer
- pviewer.html - pviewer HTML
- pviewer.js   - pviewer JavaScript
- pviewer.css  - pviewer CSS

misc - support files
- lighttpd.sh - simple script, run web server via podman for testing pages
